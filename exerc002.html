Aprendendo a adicionar imagens com HTML

CTRL + 'espaço' abre a caixa do visual studio code para adicionar a imagem

E, além disso, podemos adicionar uma url dentro do src para pesquisar direto da interner, Contudo, se o servidor onde a imagem está armazenada cair, a imagem do site cai junto.

Inserindo Icon

Escrever "link" escolher link: favicon, sintaxe: (link rel="shortcut icon" href="icone.ico" type="image/x-icon") o arquivo tem que ser ".ico"
